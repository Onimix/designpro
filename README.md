# Design Pro by Onimix

A powerful AI-driven graphic design platform — built as a single HTML file.

## Features
- 🤖 **AI Design Agent** — AutoGPT-style: Understand → Plan → Execute → Refine
- 🎨 **OpenAI-Compatible API** — Works with GPT-4o, Claude, Mistral, any OpenAI-compatible endpoint
- 📋 **13+ Templates** — Flyers, Social Media, Posters, Business Cards, Certificates, Logos
- ✏️ **Advanced Editor** — Drag/drop, resize, rotate, layers, undo/redo (50+ steps)
- 🔷 **12 Shape Types** — Rect, Circle, Star, Heart, Arrow, Hexagon, Speech Bubble & more
- 🖋 **Text Styles** — 8 font families, bold/italic/underline, letter spacing, line height
- 🖼 **Image Upload** — PNG, JPG, SVG, WebP support
- 📷 **Stock Photos** — Unsplash integration
- 🌈 **Backgrounds** — 24 solid colors, 12 gradients, patterns
- 💾 **Local Library** — Save & reload designs
- 📤 **Export** — PNG (1×/2×/3×), JPG, SVG, JSON project file

## Usage
1. Open `index.html` in any modern browser
2. Select canvas size
3. Go to **AI Log tab** → add your OpenAI API key & base URL
4. Use **AI Agent tab** to generate designs with prompts
5. Refine with commands like "Make it more modern" or "Use warmer colors"

## AI Setup
- **API Key**: Your OpenAI (or compatible) key
- **Base URL**: `https://api.openai.com/v1` (or any OpenAI-compatible endpoint)
- **Model**: `gpt-4o`, `gpt-4-turbo`, or any compatible model

## Deploy
Drag `index.html` onto [Vercel](https://vercel.com) or [Netlify Drop](https://app.netlify.com/drop) — works instantly.
