# DesignPro 🎨

A full-featured graphic design tool like PosterMyWall — built with AI.

## Features
- 12 professional templates (poster, sale, event, restaurant, fitness, etc.)
- Drag & drop canvas editor — text, shapes, images, badges, stickers
- Full properties panel — font, color, opacity, size, rotation, alignment
- Layers panel, undo/redo, zoom, grid overlay
- Export PNG / JPG / PDF
- **AI Design Generator** powered by Google AI Studio (Gemini 2.0 Flash)

## AI Design Feature
1. Open the app → click the **AI** tab
2. Paste your [Google AI Studio](https://aistudio.google.com) API key
3. Describe your design
4. Click **Generate Design** — AI creates and applies the layout instantly

## Usage
Just open `index.html` in any browser. No install needed, works offline.

## Deploy to Vercel
1. Fork this repo
2. Import into [Vercel](https://vercel.com)
3. Deploy — live in seconds!

## Built with
- [Fabric.js](http://fabricjs.com/) — canvas engine
- [Google Gemini API](https://aistudio.google.com) — AI design generation
- Vanilla JS, HTML5, CSS3
